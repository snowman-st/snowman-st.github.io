Git使用

2019-09-17
计算机

Contents

一、在Ubuntu上安装git，配置个人信息

1
2

git config --global user.name 'snowman-st'
git config --global user.email 'snowmanxk.@qq.com'

二、与git建立安全连接

首先生成公钥：

ssh-keygen -t rsa -C ‘snowmanxk@qq.com’

会生成一个公钥文件，然后在git上添加此公钥。

三、新建repository

如果本地已有一个工程，建库的时候不要勾选创建README.md文件，否则需要pull一下库里面的文件。

1
2
3
4
5

git init
git add .
git remote add origin https://github.......
git commit -m 'first commit'
git push -u origin master
