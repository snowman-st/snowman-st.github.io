Hexo 个人博客的搭建与配置

计算机

这是一篇讲解搭建个人博客的教程。此部分是摘要，后面的内容是正文部分。另外本文的编辑格式是一个标准的文章格式。

一、在github中创建一个新的repository，然后将github库与本机进行连接。

1
2
3

$ git config --global user.name 'snowman-st'
$ git config --global user.email 'snowmanxk@qq.com'
$ ssh-keygen -t rsa -C 'snowmanxk@qq.com'

​ 将生成的公钥id_rsa.pub复制到github上即可。

​ 使用：

1

ssh git@github.com

​ 进行验证，是否连接成功。

二、安装Node.js、npm、hexo

1
2
3

$ pip install node.js
$ pip isntall npm
$ npm install -g hexo-cli

三、创建初始的博客

1

$ hexo init myblog

​ 此举会生成一个名为myblog的文件夹，然后我们进入到该文件夹下面进行网站的创建和配置。

​

1
2
3

hexo new Create
hexo g
hexo s

​ 此时可以通过localhost:4000访问网站

​ 接下来，配置网站，在myblog下面的_config.yml中，加入如下配置：

​

1
2
3
4

deploy:
    type: git
    repo: https://github.com/snowman-st/snowman-st.github.io.git
    branch: master

​ 安装github的部署插件：

​

1

npm install hexo-deployer-git --save
1
2
3

hexo clean
hexo g
hexo d

​ 成功后可以通过 snowman-st.github.io访问网站

四、安装自定义风格的主题

​ 这个过程可以有，看个人执念。在调试的过程中使用:

​

1

hexo s

​ 来在本地：locahost:4000端口启动一个服务，用来实时监控自己所做的修改。

4.1 选择主题

​ 到hexo themes的网站上去挑选一个自己喜欢的风格的主题。

​ 然后就git clone到本地：

​

1

git clone git@github.com:dongyuanxin/theme-bmw.git themes/bmw

​ 然后会发现themes文件夹下面多个个bmw的主题文件夹。接下来，只需要将_config.yml中的theme设置成bmw即可。

个人认为还不错的一些主题：

​ Melody

​

4.2 外接评论和浏览服务

​ 进入leancloud.cn注册一个免费账户。leancloud是国内领先的移动服务端整体解决方案提供商。为移动应用开发者提供稳定可依赖的后端云服务，包括存储、账号管理、社交分享、推送、应用统计等以及相关的技术支持和服务。

​ 创建应用(开发版)->进入到新创建的应用下，创建两个class,一个命名为：Comment,一个命名为：Timer。Comment设置为限制写入，Timer设置为无限制。然后找到属于自己的appid和appkey,复制下来，在themes/bmw/_config.yml中进行替换。

4.3 进一步私人化

​ 此时博客所有的功能都可以正常使用了，对页面上不喜欢的功能可以在文件中的相应位置删除，或者在_config.yml中通过功能开关进行控制。
