最近实验室服务器突然出现一些问题，之前的深度学习的环境都不能用了，于是我把所有之前的环境清理掉了，重新折腾了一遍环境。

配置环境之前要搞清系统的硬件，想安装的软件版本，以及pytorch版本之间的对应关系，所以第一步查系统的信息。敲黑板，很重要！！！

零、配置静态IP

桌面版的Ubuntu 16.04很好配置，对于一台刚拿到手的纯净的Ubuntu server 的系统来说，第一步需要先让其联网。

1、查看网卡名

iwconfig 或者 ipconfig

一般查到的网卡名为： eth0/ens33

2、编辑配置文件

sudo vim /etc/network/interfaces

假设查到的网卡名为：eth0

1
2
3
4
5
6

auto eth0
iface eth0 inet static
address 192.168.31.99
netmask 255.255.255.0
gateway 192.168.31.1
dns-nameservers x.x.x.x

然后，使用

1

sudo /etc/init.d/networking restart

即可，如果此时可以ping通网关，但是不能ping通外网，可以将DNS地址改为：114.114.114.114或者8.8.8.8

一、查看系统信息

1、显卡型号

lspci -vnn | grep VGA -A 12

查出GPU是GTX 1080,根据自己想安装的cuda版本去官网下载对应的显卡驱动。

现在cuda有10.1版本了，要求NVIDIA Driver 一定要高于418

2、Ubuntu内核版本，gcc版本

uname -r

gcc –version

如果不满足要求需要及时升级。

二、安装NVIDIA驱动

1、清除之前的环境

首先查看系统中是否存在其他版本的驱动

1

sudo dpkg --list | grep nvidia-*

为避免后面的冲突，这里最好将之前的环境都清理一下

1
2

sudo apt-get autoremove --purge nvidia*
这里autoremove会将与其相关的依赖也都清楚，平时慎用，一般用remove即可。

同时清除之前不能用的cuda

1
2
3

直接运行：
sudo /usr/local/cuda-10.0/bin/uninstall_cuda_10.0.pl
sudo rm -rf /usr/local/cuda-10.0/

2、安装驱动

  • 首先，禁用nouveau

在/etc/modprobe.d/blacklist.conf中添加 ： blacklist nouveau

1

sudo update-initramfs -u

可以使用：lsmod | grep nouveau 查看是否已经被禁用，如果没有输出信息则表明已经禁用成功。

  • 其次，关闭图形界面，由于是远程操作，这一步实际上我没做。

sudo service lightdm stop

  • 最后，就可以进入到文件的下载目录了
1
2
3
4
5

sudo chmod a+x NVIDIA-LINUX-X86_64-410.78.run
sudo ./NVIDIA-LINUX-X86_64-410.78.run -–no-x-check –-no-nouveau-check –-no-opengl-files
#no-x-check 安装驱动时关闭X服务
#no-nouveau-check 安装驱动时禁用nouveau
#no-opengl-files 只安装驱动文件，不安装OpenGL文件；安装了，如果是双显卡的话，会出现重复登录的问题

安装的时候，会有配置选项，一般来说选择默认就好，仔细读一下

使用 nvidia-smi 如果可以查看信息，则成功

3、安装cuda

​ 到cuda官网下载了10.0的版本，官方下载很慢，但是服务器配置了proxychains,直接

1

proxychains wget https://developer.nvidia.com/cuda-10.0-download-archive?target_os=Linux&target_arch=x86_64&target_distro=Ubuntu&target_version=1604&target_type=runfilelocal

很快完成下载

安装过程，有一步是问是否需要再安装driver，由于已经安装，所以这一步可以选择no;其余默认

4、cuDnn

到上一步，程序已经可以在GPU上运行了

NVIDIA cuDNN是一个GPU加速深层神经网络原语库。它提供了在DNN应用程序中频繁出现的例程的高度优化的实现，所以为了更高速的执行我们的运算，需要安装cuDnn。

去官网下载对应版本的cuDnn压缩包

 1

tar -zxvf cudnn-10.0-linux-x64-v7.4.2.24.tgz
sudo cp cuda/lib64/* /usr/local/cuda-10.0/lib64/
sudo cp cuda/include/* /usr/local/cuda-10.0/include/

cd /usr/local/cuda/lib64/
sudo rm -rf libcudnn.so libcudnn.so.7
sudo ln -s libcudnn.so.7.6.0 libcudnn.so.7
sudo ln -s libcudnn.so.7 libcudnn.so 
sudo ldconfig
#查看cudnn的信息
cat /usr/local/cuda/include/cudnn.h | grep CUDNN_MAJOR -A 2

三、开发环境

anaconda 的安装，装好之后创建新的环境，然后去pytorch官网上按照指示下载与cuda版本，系统版本相应的pytorch即可。
