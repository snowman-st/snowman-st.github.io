使用hugo搭建自己的博客

由于在前两天刚刚用hexo搭建了一遍个人博客，所以搭建起hugo很轻松。hexo搭建博客可参考Hexo 个人博客的搭建与配置。

1、EXE文件安装

​ 本文介绍windows系统下的hugo博客搭建，所以首先到发行版本下载对应的压缩包，之后解压会得到一个exe文件。相较于hexo需要装node.js和npm，hugo简直不要太方便。直接在环境变量中添加相应目录即可使用hugo。

2、创建博客

​

1

hugo new site myblog

​ 选择主题，git clone到本地themes下面。将新下载的主题中的exampleSite下的config.toml复制到myblog下面。

使用：hugo server -D即可启动本地服务端口1313.

3、部署到github pages

​ 在github上建立同用户名仓库，操作方法与在hexo中建仓库的步骤一致。将themes/even/archetypes下面的default.md复制到myblog/archetypes下面，设置 draft 为false，或者直接去掉那一行。

​ 在myblog下

1

hugo new post/test.md

然后在文章里面写入内容即可。

文章写好之后，使用 hugo 命令可实现文章的编译，同时会在该目录下面生成public文件夹，进入到public文件夹下面：

1
2
3
4
5

git init
git remote add https://github.com/snowman-st.github.io.git
git add .
git commit -m 'df '
git push -u origin master

如果创建的repository里面没有README.md，使用上述命令会成功。如果已经有README.md,在push之前pull一下。

1

git pull --rebase origin master

至此，可通过 snowman-st.github.io访问本博客。

4、后续博客的更新

​ 写了个脚本up.bat：

1
2
3
4
5
6

hugo
cd public
git add .
git commit -m 'add new posts'
git push origin master
cd ..

​ 执行up即可完成新增文章的上传。
